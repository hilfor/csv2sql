=======
CSV2SQL
=======

Convert CSV data into SQL.


Features
========

- Automatic type inference.
- Customizable type deciding rules.
- Pipe usage.


Installation
============

Run ```pip install csv2sql```.


Execution
=========

For basic usage, run the command below.

.. code-block:: shell

    csv2sql all

For details, run the command with the `-h` option.
